💡환경 변화에 강인한 실시간 무인비행체 탐지/추적 알고리즘 연구

🤝🏻 팀과제  | 📅 2019.04.15 ~ 2019.11.30

    📌 주요 내용

    1. 딥러닝 탐지 네트워크의 성능 개선을 위한 드론 이미지 데이터 세트 생성
    2. 속도와 정확도를 고려하여 적합한 알고리즘을 선정하고 학습 및 평가 수행
    3. FPN Detector와 Optical Flow Tracker 연동을 위한 작업
    4. 오검출을 줄이기 위한 후처리 알고리즘 개발

    🙋‍♀️ 팀내 역할 - 학생 연구원

    1. 드론 영상 시퀀스 확보 및 Ground Truth 생성, 데이터 정리
    2. 논문 검토,머신러닝 / 딥러닝 기반 탐지 네트워크 분석 (Cascades Classifier, YOLOv3, YOLOv3-tiny, MobileNet+YOLOv3, FPN 등)
    3. Cascades Classifier, YOLOv3, FPN Train/Test, 결과 분석
    4. Boost 라이브러리를 이용하여 Visual Studio(C++)로 구현되어있는 Tracker와 Tensorflow로 구현되어있는 Detector를 연동
    5. 영상 시퀀스의 연속적인 특성(드론의 이동거리 및 크기)을 이용하여 Detector의 탐지 결과 중 오검출 판단하는 후처리 알고리즘 작성(C++)

    📚 스택



    💻 개발 환경



    CPU Intel i7-6850K 3.6Hz    |    RAM 32GB    |    GPU NVIDIA Geforce GTX 1080Ti
    python 3.6    |    Tensorflow 1.10.0    |    OpneCV 3.3.0

    • 딥러닝 탐지 네트워크 동향 및 구조 파악
    • Boost라이브러리를 이용한 구현 경험
