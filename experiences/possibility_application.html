    1. 상표 이미지의 출원 가능성을 높음/보통/낮음으로 판별하는 알고리즘 구현
    2. 출원 가능성 판별 API 서버 개발

    • 구현 과정
      • 피쳐맵을 이용한 유사도 판별은 distance 값으로 판단하게 되지만 사람이 보기에 유사한 상표일지라도 distance가 확연히 큰 샘플이 존재.
      • 따라서 단순히 distance에 임계값을 지정해 출원 가능성을 판별하는 것은 어렵다고 판단하여 OpenCV 특징점 매칭 알고리즘을 추가 도입
      • 피쳐맵 distance 및 ORB 특징점 매칭 알고리즘을 조합하여 출원가능 판별 알고리즘 구현
      • 입력 이미지의 Color Space에 따른 distance 패턴 분석 후 유동적 임계값 사용
        • 쿼리 이미지와 비교 대상의 이미지가 각각 흑백/RGB 이미지 일 경우 distance 차이가 더 크게 나기 때문

        def matching_target(img1, img2, N=50 ):
            # ORB 디스크립터 생성
        #     orb = cv2.ORB_create()
            orb = cv2.ORB_create(
                                    nfeatures= 100, # 40000, # ORB 객체가 한 번에 검출하고자 하는 특징점의 개수
                                    scaleFactor=1.2, # 스케일 계수(너무 클 경우 매칭 확률 떨어짐, 너무 작을경우 연산속도 느려짐)
                                    nlevels=8, # 이미지 피라미드를 설정. 2일 경우 이미지 크기가 절반
                                    edgeThreshold=80, #이미지 테두리에서 발생하는 특징점을 무시하기 위한 경계의 크기
                                    firstLevel=0, # 시작 피라미드 레벨 (원본이미지를 넣을 피미드 레벨)
                                    WTA_K=4,# 비교점은 BRIEF 기술자가 구성하는 비교 비트 2진 ~ 4진
                                    scoreType=cv2.ORB_HARRIS_SCORE,# 피쳐의 순위를 매기는데 사용, cv2.ORB_HARRIS_SCORE or cv2.ORB_FAST_SCORE
                                    patchSize=31,# 엣지 임계값과 상호작용하므로 엣지 임계값이 패치 크기의 값보다 커야함
                                    fastThreshold=20,
                                )
         
            gray1 = cv2.cvtColor(img1, cv2.COLOR_BGR2GRAY)
            gray2 = cv2.cvtColor(img2, cv2.COLOR_BGR2GRAY)
            
            # 특징점 검출과 디스크립터 계산
            keypoints1, descriptors1 = orb.detectAndCompute(gray1, None)
            keypoints2, descriptors2 = orb.detectAndCompute(gray2, None)
         
            # 특징점 그리기
            image1_keypoints = cv2.drawKeypoints(gray1, keypoints1, None, color=(0, 255, 0))
            image2_keypoints = cv2.drawKeypoints(gray2, keypoints2, None, color=(0, 255, 0))
         
            # 특징점 매칭기 생성
            matcher = cv2.DescriptorMatcher_create(cv2.DescriptorMatcher_FLANNBASED) # cv2.DescriptorMatcher_BRUTEFORCE_HAMMING
        # cv2.DescriptorMatcher_BRUTEFORCE
        # cv2.DescriptorMatcher_BRUTEFORCE_HAMMING
        # cv2.DescriptorMatcher_FLANNBASED
            # 특징점 매칭
            matches = matcher.match(descriptors1, descriptors2)
         
            # 거리를 기준으로 매칭 결과 정렬
            matches = sorted(matches, key=lambda x: x.distance)
         
            # 상위 N개의 매칭 결과 선택
            best_matches = matches[:N]
         
            # 매칭 결과 그리기
            result = cv2.drawMatches(img1, keypoints1, img2, keypoints2, best_matches, None)
